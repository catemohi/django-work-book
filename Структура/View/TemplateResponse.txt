Класс TemplateResponse нужен для формирование ответа клиенту.
В отличии от HttpResponse, поддерживает отлежненный рендеринг шаблона.
Рендеринг при использовании TemplateResponse, выполняется полсле прохождения всей цепочки зарегистрированных Middleware.
Тоесть непосредственно перед отправкой клиенту.
(Дронов В. А. - Django 3.0. Практика создания веб-сайтов на Python-БХВ-Петербург (2021) | 186стр. 9.3.3. Класс TemplateResponse: отложенный рендеринг шаблонов )

Методы 
TemplateResponse(request, template)
Обязательные аргументы:
* request - экземпляр запроса.
* template - шаблон, который будет передан клиенту.
Можно влиять на контекс с помощью именновоного аргумента context
(https://django.fun/docs/django/ru/4.0/ref/template-response/#id1)
