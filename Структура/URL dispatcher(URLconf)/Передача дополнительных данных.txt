Функция path может принимать на вход дополнительный аргумет.
С помощью этого аргумента, который ожидает на вход словарь, можно передовать дополнителньые параметры.
( https://docs.djangoproject.com/en/4.1/topics/http/urls/#passing-extra-options-to-view-functions )
Дополнительные параметры view сможет получить также через одноименные параметры. 
Если в маршруте присутствует URL-параметр с тем же именем, что и у дополнительного значения, передаваемого через словарь, view будет передано значение из словаря. Извлечь значение URL-параметра перезапишется.
