Имена приложений запписываются в файлах urls на уровне вложенности urlpatterns

Пример:

app_name = ' bboard '
urlpatterns = [
        ...
    ]

Имена приложений применяются для предотвращения ошибки с пересечением шаблонов.


Чтобы сослаться на маршрут, объявленный в нужном приложении, следует предварить имя маршрута именем этого приложения, разделив их двоеточием.

Примеры:

url = reverse ('bboard:by_rubric', kwargs= { 'rubric_id': 2))
Результат : /bboard/2/

{% url 'bboard:index' %}
Результат : /bboard/
